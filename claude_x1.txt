  📦 Tahap 1 — Push ke GitHub (kalau belum)

  2. Di terminal (ganti USERNAME):

  cd "D:/Productifity/Coding/Keyst"
  git remote add origin https://github.com/USERNAME/keyst.git
  git branch -M main
  git push -u origin main

  ▎ Kalau tadi sudah pernah git remote add, ganti barisnya jadi git remote 
  ▎ set-url origin https://github.com/USERNAME/keyst.git.

  ---
  🐘 Tahap 2 — Buat database gratis di Neon

  1. Daftar di https://neon.tech (bisa login pakai GitHub, tanpa kartu kredit).
  2. Create Project → beri nama keyst → pilih region terdekat (mis. Singapore).
  3. Setelah jadi, muncul Connection string. Pilih yang Pooled connection (ada
  tulisan Pooled), lalu salin. Bentuknya seperti:
  postgresql://user:pass@ep-xxxx-pooler.ap-southeast-1.aws.neon.tech/neondb?sslm
  ode=require
  Simpan dulu — dipakai di tahap berikutnya.

  ---
  ▲ Tahap 3 — Deploy ke Vercel

  1. Daftar di https://vercel.com → login pakai GitHub.
  2. Add New… → Project → Import repo keyst.
  3. Sebelum klik Deploy, buka Environment Variables, tambahkan:
  - Name: DATABASE_URL
  - Value: (tempel connection string Neon dari Tahap 2)
  4. Klik Deploy.

  ---
  ✅ Yang terjadi otomatis

  Berkat script vercel-build yang kubuat, saat deploy Vercel otomatis
  menjalankan prisma db push → tabel database dibuat sendiri di Neon. Database
  mulai kosong, siap kamu isi latihan. Selesai build, kamu langsung dapat URL
  live seperti keyst.vercel.app.

  Setiap git push ke depannya → auto-redeploy. 🎉

  uire

  Pastikan saja:
  - ✅ diawali postgresql://

  dipakai sehari-hari.
